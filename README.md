# Phone-stand/ tablet stand
I made this phone stand and i extened it so my ipad also fits on it and doesn't fall off. There is a hole in the back for the charger to pass through.

Update: 
1. I added a small box for paper clips on the bottom of the phone stand
2. I added a headphone stand for my headphone. It can be attached to the main frame using existing poles.
3. I added a pencil box which has an interesting lock. If you don't use the lock you can hang your keys on it.
